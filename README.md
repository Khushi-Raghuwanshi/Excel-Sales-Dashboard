# Sales Dashboard using Excel

## Overview
An interactive sales dashboard created using Microsoft Excel to analyze sales trends, seller performance, and city-wise sales.

## Features
- Pivot Tables
- Pivot Charts
- KPI Cards
- Slicers for dynamic filtering
- Monthly Sales Analysis
- Seller Performance Analysis
- City-wise Sales Analysis

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
